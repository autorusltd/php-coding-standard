## Installation

```bash
composer require arus/php-coding-standard
```

## How to use?

```xml
<?xml version="1.0"?>
<ruleset name="Arus Coding Standard">
    <rule ref="./vendor/arus/php-coding-standard/ruleset.xml"/>

    <file>src</file>
    <file>tests</file>
</ruleset>
```
